## Como testar 🚀

• Clone esse repositório ou baixe como .zip <br>
• Execute os seguintes comandos dentro da pasta clonada

```sh
 yarn
```

```sh
 yarn dev
```

• Abra `localhost:3000` em seu navegador e divirta-se 😊
